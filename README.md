## Features

- Terminal-inspired UI with authentic command-line feel
- Matrix-style rain animation background
- Responsive design with mobile support
- Real-time command prompts
- Modular window system
- LinkedIn integration
- Zero dependencies

## Tech Stack

- HTML5 Canvas for Matrix animation
- Pure CSS3 for terminal styling
- Vanilla JavaScript for interactions
- GitHub Pages for hosting
